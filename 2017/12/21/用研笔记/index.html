用户研究概述

==用户研究员的能力==，技术技能和非技术技能

技术技能

  1. 用户研究能力
  2. 人机交互能力
  3. 产品分析能力

一，用户研究能力

针对产品开发的不同阶段，能够灵活运用各种调研方法，权衡利弊，制定适合的用户研究方案

能够输出高质量的用户调研报告，为产品设计提出合理化建议

用户研究能力包括：

  1. 定量分析能力
  2. 定性分析能力

定量分析能力：

  1. 数据的敏感度
  2. 数据分析能力
  3. 可熟练使用各类定量研究方法
  4. 掌握各类定量研究方法的应用场景及优缺点
  5. 独立完成分析的能力

定性分析能力：

  1. 对常见定性研究方法，可熟练使用
  2. 对各个研究方法有自己的心得体会，掌握技巧
  3. 掌握各类定性研究方法的应用场景及优缺点
  4. 独立执行研究的能力

二，人机交互能力

  1. 熟练掌握人机交互的理论知识
  2. 熟悉掌握各终端设计规范
  3. 有良好的用户体验意识
  4. 可通过用户研究，发现产品设计中的不足，并能提供设计改善建议

三，产品分析能力

  1. 对所应聘行业非常了解，有充分准备
  2. 对所应聘行业的竞品非常熟悉
  3. 对所应聘部门的产品非常熟悉，并有自己的见解
  4. 有良好的产品感
  5. 热爱互联网产品，对于新兴产品有好奇心，保持对互联网产品的关注和体验

非技术技能

  1. 沟通技巧
  2. 好奇心
  3. 学习能力
  4. 敏锐力

选择定性还是定量

取决于

  1. 用户画像的受众是谁？设计师?产品经理？
  2. 希望用户画像结果来支持什么类型的决定
  3. 打算花多少时间和钱

用户画像的特点

  1. 越早越好，方向准确比行动重要
  2. 任何阶段都可以，任何阶段都有用，迟到比旷工好
  3. 不断迭代更新，用户会变化
